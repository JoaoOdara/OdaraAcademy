# Odara Academy

Plataforma de trilhas de conhecimento e microtreinamentos — Odara Alfajores.

## Arquivos deste repositório

| Arquivo | O que é | Precisa subir no GitHub? |
|---|---|---|
| `index.html` | A plataforma completa — abre direto no navegador | ✅ Sim |
| `supabase-schema-v1.sql` | Schema inicial do banco (tabelas base) | ✅ Sim (referência) |
| `supabase-migration-v2.sql` | Migração v2 (matriz, facilitadores, notificações) | ✅ Sim (referência) |

## Como usar

1. Rode `supabase-schema-v1.sql` no Supabase SQL Editor
2. Rode `supabase-migration-v2.sql` logo em seguida
3. Abra `index.html` no navegador — já conecta ao banco automaticamente
